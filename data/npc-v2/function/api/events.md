# Event types
- on_left_click
- on_right_click
- on_tick
- on_spawn
- post_spawn


register events for type under function tag:
```
npc-v2:events/$TYPE/$EVENT_TYPE
```

# Player event types

# player events

```
npc-v2:events/$TYPE/player/$EVENT_TYPE
```
